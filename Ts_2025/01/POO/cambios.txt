¿Cuál es el problema?

El navegador no entiende require() ni module.exports, ya que son parte del sistema de módulos CommonJS (usado en Node.js), no del navegador.

Tu código compilado contiene esto:

var Empleado_1 = require("./Empleado");


Eso está bien para Node.js, pero no funciona en HTML con <script>.


Solución: Compilar con módulos ES6 (usando import y export que sí entiende el navegador)

{
  "compilerOptions": {
    "target": "ES6",
    "module": "ES6",
    "moduleResolution": "node",
    "outDir": "./dist",
    "rootDir": "./",
    "strict": true
  }
}
Aplicar 
tsc main.ts --module ES6 --target ES6 --outDir dist
